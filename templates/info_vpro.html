Versión Pro - OutVirus


Características de la Versión



- Esta versión se configura completamente desde consola

- Su interfaz es mediante consola de comandos

- Se permite registrarse con usuario de github e introduciendo su página web

- Para usar esta versión son necesarios conocimientos básicos de Python

- Se requiere saber usar la Terminal de Windows

- Es necesario saber ejecutar programas desde la Terminal

- Se recomienda tener conocimientos básicos de programación

- Se requiere que el equipo donde va a ser ejecutado el programa sea de 64 bits y posea Windows



Instrucciones de uso y configuración de esta Versión


1. Inicie el programa portable, tras haberlo descargado en el link al final de la pagina: 'mainvpro.exe'

2. El programa se abrirá en terminal

3. En la primera flecha introduzca el idioma con el que desee configurar el programa (es, ing, fr)

4. En la segunda flecha se configura el nombre de usuario (Opcional)

5. En la tercera flecha se configura su correo electrónico (Opcional)

6. En la cuarta flecha se configura su página web como desarrollador

7. En la quinta flecha se configura su usuario de Github (Opcional)

8. Espere hasta que finalicen los procesos de configuración y se abrirá una terminal

9. Comandos:

- re - revisión del equipo

- rc - revisión completa del equipo

- exit - cerrar el programa
